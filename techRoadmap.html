Tech

Exchange of LunarScape gold and $LUNAR

LunarScape gold ➔ $LUNAR

In game, players can choose to exchange a portion of their in-game gold for $LUNAR. After they have performed this action, their in-game gold will be reduced and this data will be exposed by an API endpoint. This API will be periodically queried by our custom oracle service running on AWS infrastructure leveraging the web3.js library to communicate with the Binance Smart Chain via QuickNode. Once the oracle reads the data, it will initiate a transaction with the $LUNAR token smart contract to update the player's nominated balance.

$LUNAR ➔ LunarScape gold

To perform the reverse action, we will provide an in-browser dashboard, again leveraging the web3.js library as well as MetaMask to allow players to initiate a transaction with the $LUNAR token smart contract to burn a nominated amount of $LUNAR. The game will periodically query the smart contract and update in-game gold accordingly.
